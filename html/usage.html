Tool Usage Help

This is a tool to find Match ID from Riot ID.
Originally intended for tournaments using Tournament-V5 of the Riot Games API on the live server.
The tool is programmed by BeeBee. If you have any problems with the code, you can find me (beebeespc) at discord Leaguepedia.

Player Match Tracking

Field: Riot ID
This is a required field. And note, please write down the current player's Riot ID, not the summonername or old Riot ID. And you must fill in the Riot ID#Tagline, otherwise the software will not work.
If you don't know the Riot ID but only know the name on the display of the match HUD then you can use op.gg, u.gg and tracker.gg to find them and if it gives many results try with all of them.

Field: Start Date and End Date
Time here is the time frame you want to retrieve information from, not the time of ONE match
Both of these fields are optional, but when entering, please enter both the date and time because if you only enter one of the two, it will return an empty result.

Field: Match Type
The default for this field is Tourney (Games created with Tournament Code)
but if you type in a player's Riot ID and it doesn't work and you're wondering if you've entered something wrong, I've added an option called All to see all of their games.

List of Match

(1) This is the exported Match ID you can click on to go to the match fetch page
(2) This is the champion the player uses in the match.
(3) Is the time the Lobby was created. The top row is UTC +0 and the bottom row is your Local Time
(4) Is a list of players who participated in that match (Names will be based on their names at the time of playing that match, not their names at the current time)

Previous - Next (Button)
You can only access 10 matches at a time so just use it to watch more but I think if you use Starttime and Endtime properly you won't need it hehe :D

Match Data

Field: Match ID
This field is required and please enter the match code in the following Format {Servercode}_{MatchID}.

Invalid {Servercode} includes Special Realms such as Development Realm or Tournament Realm or Tencent and Garena Clients
Valid {Servercode} includes:
NA1 North America
BR1 Brazil
LA1 Latin America North
LA2 Latin Ameraca South
EUN1 Europe North East
EUW1 Europe West
ME1 Middle East
TR1 Turkey
RU1 Russia
JP1 Japan
KR1 South Korea
OC1 Oceania
PH2 Philippines
SG2 Singapore, Malaysia and Indonesia
TH2 Thailand
TW2 Taiwan, Hongkong, Macao
VN2 Vietnam

{MatchID} you can get with my Player Tool or you can get in Data:{Tournament Name} of Leaguepedia or in the match history section in the image below

Match Details

(1) Is the number of kills on both sides
(2) Is the total time of the game
(3) Is the Player's Champion Icon (if the Icon cannot be found it will be replaced by the champion name)
(4) It is Ingame when that player plays that match
(5) It is GMT and your local time
(6) The map where the match takes place
(7) The patch that the match uses (if hovered over it will show the full name of that patch as shown below)

Player Riot ID List (Current)
This is the box containing the current names of all players in that match, it will be arranged by P1 Blue Team ⮞ P5 Blue Team ⮞ P1 Red Team ⮞ P5 Red Team


When you click on a name you will be taken to the match page using the riot ID.

Why I created this?

This is all for the purpose of simplifying the way to get Match IDs from tournament matches on the Live server for Leaguepedia and be able to add them into Data:{Tournament Name} as in the example below.

You can use Leaguepedia's Parser to convert the Match ID into the Scoreboards code on lol.fandom.com
